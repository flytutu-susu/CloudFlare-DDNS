DDNS Service Notification - Program has been automatically restarted

Hello,

Your CloudFlare DDNS service has an exception because you have configured the auto_restart option, which has automatically restarted the CloudFlare DDNS service for you. The exception is not necessarily triggered by the DDNS service itself. It may also be caused by the busy api of CloudFlare or the failure of services such as speedtest.

You can connect to the server to view the exception details through the log/DDNS.log file. Please troubleshoot the cause and check whether the DDNS service has been restarted successfully, so as to prevent your server from being unable to be accessed through the domain name due to IP changes.
